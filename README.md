# LanspammerPython
Инструмент для создания фейковых серверов
они будут отображатся в списке локальных миров
другие игроки не могут зайти их просто кикнет
также есть режим голосования он добавляет к каждому названию сервера счетчик голосов
и при заходе на фейк сервер он кикнет тебя а к счетчику добавится один
